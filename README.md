# Harness Lab

Mini Harness Engineering system in Python, based on Markdown contracts.

## Quick Start

```bash
pip install click pytest black isort flake8
python cli.py run
```

## Project Structure

- `contracts/` - Sprint contracts (Markdown)
- `harness/` - Engine (loader, runner, evaluator, reporter)
- `src/` - Application source code
- `tests/` - Tests
- `reports/` - Generated reports
